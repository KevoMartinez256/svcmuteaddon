Coloca aquí tu icono del mod como 'icon.png'.

Recomendaciones:
- Formato: PNG con transparencia (opcional)
- Tamaño sugerido: 256x256 o 512x512 px (cuadrado)
- Nombre exacto: icon.png
- Ruta final en el JAR: assets/svcmuteaddon/icon.png

Esto se referencia desde META-INF/neoforge.mods.toml con:
logoFile="assets/svcmuteaddon/icon.png"

Tras copiar el archivo, ejecuta:
  ./gradlew runClient
para validar que aparece en la lista de mods.
